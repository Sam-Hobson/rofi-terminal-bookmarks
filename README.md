# rofi-terminal-bookmarks
Access, add, and delete terminal bookmarks through Rofi! Use to easily access favourite directories.

## Setup

1. Clone this repo:
> git clone https://github.com/Sam-Hobson/rofi-terminal-bookmarks.git

2. Change the directories in each file as advised.

3. Move (and potentially rename) bookmark_add.sh and bookmarks_del.sh into your PATH so they can be easily accessed.

4. Create a keyboard shortcut to execute bookmarks.sh, and quick_bookmark.sh.

5. Enjoy easily opening terminals in all your favourite directories!
